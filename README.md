# Blinkit Grocery Sales & Delivery Analytics Dashboard

## Project Overview

This project analyzes Blinkit grocery sales and delivery performance using SQL and Power BI. The goal is to transform raw business data into meaningful insights through data analysis and interactive dashboard visualizations.

## Objectives

- Analyze overall sales performance.
- Track yearly sales trends.
- Evaluate delivery performance.
- Identify top-selling products and categories.
- Create an interactive dashboard for business reporting.

## Tools & Technologies

- SQL (MySQL)
- Power BI
- Microsoft Excel

## Key Insights

- Analyzed total sales and customer orders.
- Measured delivery status and on-time delivery percentage.
- Identified top-performing product categories.
- Visualized yearly sales trends.
- Built an interactive dashboard for business decision-making.

## Dashboard Features

- Sales Trend Analysis
- Category-wise Sales
- Delivery Performance Analysis
- Interactive Filters
- Business KPI Dashboard
